Langkah Langkah Menggunakan Website Peminjaman

Langkah 1

Untuk mengakses Transaksi, dan Profil, user harus login dengan cara klik menu "Log in"

Langkah 2

Setelah User login, maka user dapat mengakses halaman Transaksi dan Profile

Langkah 3

Pada halaman Transaksi user dapat melakukan pengisian form Transaksi, dengan cara klik "Simpan"

Langkah 4

Setelah klik "Simpan" user dapat klik "Daftar Transaksi" untuk melihat history form pengisian

Langkah 5

Setelah user selesai di halaman Transaksi, user dapat kembali ke beranda, lalu klik halaman profile

Langkah 6

Di halaman Profile ini, user dapat membaca informasi terkait website ini, dan juga user dapat memberikan feedback terkait dengan website
